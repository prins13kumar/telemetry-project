# Telemetry Data Conversion Project

## 📌 Description
This project converts two different telemetry data formats into a unified format using Python.

## ⚙️ Features
- Handles multiple JSON input formats
- Converts ISO timestamps to milliseconds
- Ensures consistent output format
- Includes automated test validation

## 🛠 Technologies Used
- Python
- JSON
- GitHub

## ▶️ How to Run
1. Clone the repository
2. Run:
   python main.py

## ✅ Output
All tests pass successfully, ensuring correct data transformation.
